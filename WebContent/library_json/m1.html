SUNBA선바


소개

안녕? 내 이름은 선바!

특징

  • 별명, 줄임말이 정말 많다.
  • ex1) 바순, 바돌 => 선바 구독자 애칭. 성별에 상관없이 고르고 싶은 걸 고르면 된다고 한다.
  • ex2) 자낳괴 => 트위치의 스트리머를 후원하는 행위를 선바가 매우 좋아하는 걸 보고 붙인 별명
  • ex3) 선또죽(선바 또 죽었다), 겜잘바(게임 잘하는 선바) => 게임 방송하는 스트리머 치고는 게임을 별로 못하는 선바를 위해 생긴 별명들.
    계속 죽다가 한 번 잘하면 '겜잘바'라고 불러주는 댓글창을 볼 수 있다.

웃음(인기) 요소

  • 놀리기 좋은 긴 얼굴 => 노각오이
  • 말 하나하나에 트집잡으면서 웃기는 유머감각
  • 남자 스트리머, 유튜버에서 보기 드문 건전하고 깨끗한 언어와 콘텐츠
  • 다양한 콘텐츠 (↓ 에서 자세한 설명)
  • 시청자에게 놀림 당하면서 부끄러워하는 모습(귀여워)

인기 동영상

주 컨텐츠

  • 게임 => Just Dance, 스팀 게임, 쯔꾸르 게임, 배틀 그라운드, 쥬니버 네이버 게임(현재 서비스 종료) 등
  • 먹방 => 빨리 먹는 게 특징, 댓글창에서는 공격적이라며 무섭다는 의견이 많다.
  • 챌린지 => ~참기, ~안 쓰기 등의 영상들. 실패 각이 서면 그때부터 정말 재미있다.
  • 시청자 참여 => 선바 코스프레 대회 등. 선바의 반응과 시청자들의 유머 감각이 유튜브 각을 좌우한다.
  • 음악 => 주로 술주정이나 긴 방송 중 노래 부른 몇 분을 칭한다. 특히 선술잘(선바는 술을 잘마시지)가 유명하다.
